# 🧰 Pack de Programas - Pós Formatação

Aplicação desenvolvida para Windows com o objetivo de facilitar a instalação de programas essenciais após a formatação do computador.

## 🚀 Sobre o Projeto

O **Pack de Programas** é uma ferramenta com interface gráfica simples, intuitiva e autoexplicativa, que permite instalar rapidamente diversos softwares essenciais em poucos cliques.

Ideal para técnicos de informática, usuários avançados ou qualquer pessoa que deseja agilizar o setup inicial do sistema operacional.

---

## 🎯 Objetivo

Após formatar um computador, é comum precisar instalar vários programas manualmente. Este projeto automatiza e centraliza esse processo, proporcionando:

- ⏱️ Economia de tempo
- 🧩 Facilidade de uso
- 📦 Centralização de softwares essenciais

## ▶️ Como usar

1. Clone o repositório:
   git clone https://github.com/TAKKiN7/pack-de-programas.git

2. Acesse a pasta:
   cd pack-de-programas

3. Execute:
   python main.py

> ⚠️ Execute como administrador.
